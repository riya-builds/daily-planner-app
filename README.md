# Daily Planner App

A productivity-focused Progressive Web App (PWA) built using HTML, CSS, and JavaScript.

This app helps users manage daily tasks, track water intake, and monitor progress with reminders and offline support.

---

## Features

- Add daily tasks
- Edit tasks
- Delete tasks
- Task completion tracking
- Search tasks
- Task priority system
- Progress bar
- Water intake tracker
- Dark mode
- Reminder notifications
- Local storage support
- Installable PWA
- Offline support using Service Worker

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PWA (Manifest + Service Worker)
- Local Storage API

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/riya-builds/daily-planner-app.git
```

2. Open the project folder

3. Run with Live Server in VS Code
